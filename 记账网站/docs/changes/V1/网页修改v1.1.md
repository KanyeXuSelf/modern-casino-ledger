         3. 需要有一块部分记录每位player最后是否结账，结了多少，还剩多少，以及和哪位合伙人结账的
      3. 记账界面还需要有一块区域反应这个局的收益部分

            1. 成本部分

               2. 每个合伙人的成本部分
            2. 净利润以及每个合伙人分到的钱（计算公式：净利润=总Buyin-总Cashout-所有成本）
